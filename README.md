这是一个用Go语言开发的辅助库，尤其适用于熟悉PHP内置函数的开发者，将会实现PHP内置函数，让你写go如php般流畅，你值得拥有~

### 下载安装

```shell

go get -u github.com/php2go/php2go/php

```

### 关于命名

PHP下划线命名转为Go大驼峰命名。

### Example:

```go

package main

import (
    "github.com/php2go/php2go/php"
)

func main() {

    php.Echo("Hello ", "world!", " This is php2go library.")

}

```
